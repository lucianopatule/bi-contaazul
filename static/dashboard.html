Passo 1 — Dados da Empresa

Informe os dados básicos do cliente que você vai cadastrar no BI

Passo 2 — Autorização no ContaAzul

O cliente precisa autorizar o acesso aos dados dele

Ao clicar no botão abaixo, uma nova aba do navegador vai abrir com a tela de login do ContaAzul.

Importante: Peça ao seu cliente para fazer login com o usuário e senha do ContaAzul dele (não o seu). Após o login, ele será redirecionado para uma URL — essa URL é o que precisaremos no próximo passo.

Passo 3 — Colar URL de Retorno

Após o login, copiar a URL inteira do navegador

Depois que seu cliente fizer login e autorizar, ele será redirecionado para uma página do ContaAzul. A URL na barra do navegador é o que importa — ela contém o código de autorização.

Instrução para o cliente: "Clique na barra de endereço, aperte Ctrl+A para selecionar tudo, Ctrl+C para copiar, e me envie essa URL por mensagem".

Passo 4 — Sincronizar Dados

Buscar tudo do ContaAzul e importar para o BI

Agora vamos trazer categorias, vendas, contas a receber e contas a pagar do ContaAzul. Esse processo leva cerca de 30 segundos.

Passo 5 — Visualizar Dashboard

Tudo pronto! Dados importados e disponíveis

Clique no botão abaixo para ir direto ao dashboard da empresa recém-cadastrada. Você verá o resumo executivo, fluxo de caixa, contas a receber e a pagar, DRE e vendas.
